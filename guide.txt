    2 - el orden de las propiedades dentro de las TAGS o clases son:
    {
        propiedades para todas los tamaños
        propiedades para dispositivos moviles  @QUERYS
        propiedades para tablets y destoks     @QUERYS
    }
    3 - si no encuentras una TAG con sus propiedades puede
    ser por que comparte propiedades con otros componentes por lo que
    hemos definido un archivo para reutilizacion de codigo entre componentes
    RUTA: 'assets/sass/main.scss'
    4 - cada TAG o clase puede contener una ligera descripcion de sus propiedades
    5 - este archivo puede contener clases añadidas con javascript
*/
